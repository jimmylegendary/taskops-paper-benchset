# TaskOps Work Graphs

This directory contains generated TaskOps work roots for benchmark execution.

The committed `core/` work graph is a ready-to-inspect example generated by:

```bash
python3 scripts/taskops_bench.py init-work \
  --work-dir taskops-work/core \
  --mode core \
  --arms both \
  --run-id core-qwen3_6_27b-001 \
  --force
```

Validate it:

```bash
taskops validate taskops-work/core
taskops summary taskops-work/core
taskops queue sync taskops-work/core --json
taskops queue list taskops-work/core --json
```

The SQLite queue under `.taskops/` is intentionally ignored. It is a rebuildable
execution projection, not canonical benchmark truth.
